정보보안 전문가

20807 백선우

정보보안전문가는 정보시스템과 정보자산을 보호하기 위해 보안정책을 수립하고, 시스템에 대한 접근 및 운영을 통제하며, 침입 발생시 신속히 탐지하여 즉각적으로 대응·복구한다

다양한 전문가

  1. 모의해킹 전문가
    • 설명: 기업의 중요 정보 시스템을 대상으로 모의해킹을 실시하여 기술적 보안 취약성 진단을 해주는 전문가
    • 기초능력: C, Python 프로그래밍, 컴퓨터 구조(CPU/Memory) 기초지식, TCP/IP 이론, Database 기초지식, OSI 7 계층
    • 필요능력: WEb/WAS 서버 해킹, 시스템 해킹(BOF/FSB), DBMS 해킹, 리버스 엔지니어링
    • 관련회사: SK 인포섹, 타이거 팀, 라온시큐어, 티오리
    • 기초 공부에 도움이 되는 추천 사이트
      • BOF 기초 달고나 문서
      • Codegate 문제풀이
      • 웹해킹 훈련 사이트
  2. 정보보안 시스템 구축엔지니어
    • 설명: 정보보안 솔루션 업체 개발자들이 개발한 S/W 제품을 기업의 수요에 맞게 현장에 가서 구축을 해주는 엔지니어
    • 기초능력: Linux, Window Server 기초지식, Database 기초지식, TCP/IP 이론, OSI 7 계층, Network Subnetmask 지식
    • 필요능력: 원할한 의사소통 능력, 강한 멘탈
    • 관련회사: 시만텍(DLP), SoftCamp(DRM), 지니안(NAC)
  3. 정보보안 솔루션 개발자
    • 설명: 정보보호 관련 사고를 미연에 방지하기 위해서 보안이 필요한 분야에서 요구되는 소프트웨어 프로그램을 개발하는 전문가
    • 기초능력: C, JAVA, JSP, Database, WEB/WAS, Linux, Window Server
    • 필요능력: 깊이 있는 JAVA 능력, 그 이외에 프로그래밍
    • 관련회사: 안랩, 시만텍(DLP), SoftCamp(DRM), 지니안(NAC)
  4. 정보보안 종합 컨설팅 전문가
    • 설명:기업의 정보자산과 비즈니스 프로세스에 따른 위협 및 취약점을 분석하여 보안 수준을 파악하고, 요구 수준에 맞는 통합적인(기술+관리)보안 해결책을 설계하는 전문가
    • 기초능력: Linux 기초지식, Window server 기초지식, 개인정보보호법, 정보통신망법
    • 필요능력: 원활한 의사소통 능력, 꼼꼼한 문서작업 능력, 발표능력, 대인관계 능력
    • 관련회사: SK 인포섹, 안랩, 이글루 시큐리티
    • 기초 공부에 도움이 되는 추천 사이트
      • Kisa 법령 현황
  5. 정보보안 침해대응 전문가
    • 설명: 기업의 보안사고가 방생했을때 피해규모를 최소화하기 위해 사고를 보호하고 시스템을 복구하고 예방전략을 수립하는 일을 하는 전문가
    • 기초능력: TCP/IP 이론, Linux 기초지식, Window Server 기초지식, 파일 구조
    • 필요능력: 패킷 분석 능력, WireShark 사용능력, Forensic 기초지식, 원활한 의사소통 능력
  6. 악성코드 분석 전문가
    • 설명: 새로운 악성코드를 분석하여 감염 경로나 방법-증산-치료 방법 등을 개발하고 치료할 수 있는 백신 프로그램을 제작하는 전문가
    • 기초능력: C, C++, 어셈블리어
    • 필요능력: 시스템 분석능력, 네트워크 트래픽 분석, 동적 및 정적 분석 툴에 대한 지식
    • 관련회사: 안랩, 이스트소프트
  7. 디지털 포렌식 전문가
    • 설명: 정보자산을 위협하여 보안사고를 발생시키는 요인에 대하여 증거물을 수집하여 복구하고 추적하는 활동을 수행하는 전문가
    • 기초능력: 컴퓨터 구조(CPU/Memory) 기초지식, TCP/IP 이론, Database 기초지식
    • 필요능력: Encase 사용 및 분석능력, FTK Imager 사용 및 분석능력, HxD 사용 및 분석능력
    • 관련회사: Plainbit, KISA, 검찰청, 사이버수사대, 해양경창철, 관세쳥, 국방부
    • 기초 공부에 도움이 되는 추천 사이트
      • 디지털 포렌식 첼린지 사이트
      • 디지털 포렌식 공부
  8. 정보보안 기술/관리적 보호조치 담당자
    • 설명: 기업의 정보 자산 보호를 위해 기술적 보호조치 및 관리적 보호조치를 취하는 담당자
    • 기초능력: 보안 솔루션 도입 및 기획 능력, F/W 운영 지식, 다양한 보안 솔루션 운영 지식, ISMS-P 인증 지식, 개인정보보호법, 정보통신망 법률 지식
    • 필요늘력: 벤더사에게 휘둘리지 않을 정도의 다양한 기술기반 보안 지식
    • 관련회사: 대기업
